# For Loop

```
for init; condition; iter:
    expressions
```

```
for item : collection:
    expressions
```

```
func main():
    for i = 0; i < 10; i++:
        >> f"{i}\n"
```

```
func main():
    for i : range(10):
        >> f"{i}\n"
```
