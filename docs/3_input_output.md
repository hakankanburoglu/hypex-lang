# Input/Output

The input function is defined with the `<<` operator and the output function with the `>>` operator.

```
func main():
    var:string name
    >> "Enter your name : "
    << name
    >> "Hello, " + name + "!"
```

Alternatively, it can be written as output in string format.

```
>> f"Hello, {name}!"
```
