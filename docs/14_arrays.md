# Arrays

```
func main():
    var:int[] nums = [2, 4, 6, 8, 10]
    for i = 0; i < nums.len; i++:
        >> nums[i] + "\n"
```

```
func main():
    var:int[] nums = [2, 4, 6, 8, 10]

        >> item + "\n"
```
