# Switch Statement

```
switch expression:
    case value:
        expression
    case value:
        expression
    ...

    _:
        expression
```

```
func main():
    var:char digit
    >> "Enter a digit : "
    << digit
    switch digit:
        case '0': >> "Zero"
        case '1': >> "One"
        case '2': >> "Two"
        case '3': >> "Three"
        case '4': >> "Four"
        case '5': >> "Five"
        case '6': >> "Six"
        case '7': >> "Seven"
        case '8': >> "Eight"
        case '9': >> "Nine"
        _: >> digit + " is not digit"
```
